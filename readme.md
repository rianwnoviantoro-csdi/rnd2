## Run Locally

Clone the project

```bash
  git clone https://github.com/rianwnoviantoro/rnd2.git
```

Go to the project directory

```bash
  cd rnd2
```

Install dependencies

```bash
  npm install or yarn install
```

Start the server

```bash
  npm run dev or yarn dev
```
